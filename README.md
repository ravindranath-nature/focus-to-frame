# Focus to Frame — Photography Website

Portfolio website for **Geethu Nair** | [@focustoframe](https://www.instagram.com/focustoframe/)

## Deploy to GitHub Pages

1. Go to [github.com](https://github.com) and create a new repository named `focus-to-frame`
2. Upload all files from this folder into the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at `https://YOUR-USERNAME.github.io/focus-to-frame/`

## Pages
- `index.html` — Home
- `gallery.html` — Full photo gallery with filter & lightbox
- `about.html` — About Geethu Nair
- `services.html` — Services & pricing
- `contact.html` — Contact form & FAQ
